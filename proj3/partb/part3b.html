B.1 Corner Detection

Corner selection can be treated as identifying pixels whose local gradient structure is two-dimensional. The Harris response operationalizes this idea by scoring locations where intensity changes strongly along two independent directions, producing a dense set of candidate interest points.

Raw Harris detections typically cluster in high-texture regions, because nearby pixels can share similarly high scores. Adaptive Non-Maximal Suppression (ANMS) converts this clustered set into a spatially diverse subset by retaining points that remain locally strong while maximizing their separation from stronger neighbors. The resulting distribution improves downstream matching because correspondence hypotheses are spread across the overlap rather than concentrated in a single patch.

The qualitative difference between the two methods is distributional rather than semantic: Harris identifies many correct “corner-like” locations but over-represents regions with repeated micro-texture, whereas ANMS explicitly trades quantity for coverage. This trade improves robustness in later stages because descriptor matching is less sensitive to a single textured area dominating the correspondence set.

The main failure mode of ANMS is under-representation in low-texture regions: if few points have high Harris scores, the suppression radius can eliminate potentially useful candidates. Practical behavior therefore depends on coupling ANMS parameters (target count and suppression criterion) to the scene and to the descriptor scale used downstream.
